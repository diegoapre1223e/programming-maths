About

Esta base está pensada para que escribas entradas en Markdown sin tocar HTML de cada post.

Estructura rápida

  • /posts/index.json: índice de entradas.
  • /posts/*.md: contenido de cada post.
  • /posts/scripts/*.js: scripts opcionales por post (interactividad).
  • /assets/js: lógica del sitio (listado, parseo Markdown, render de post).

Preparado para visualizaciones

Puedes cargar librerías 2D/3D (como Three.js) desde un script de post sin cambiar la arquitectura.

Arquitectura clara y extensible para aprender
